# Banana — Terrain

Banana features a full terrain system that adds depth and realism to the railway landscape.

## Heightmap

The terrain is built on a heightmap, where each point on the map has an elevation value. This allows for hills, valleys, and flat plains.

## Elevation Bands & Hypsometric Tinting

Different elevation levels are rendered with distinct colors using hypsometric tinting, giving a topographic map feel.

## Hillshading

Adjustable sun angle controls hillshading, providing a 3D-like appearance to the terrain through light and shadow.

## Contour Lines

Contour line overlays can be toggled on to visualize elevation changes across the map.

## Tunnels & Occlusion

- **Occlusion meshes** hide tracks that pass underground
- **X-ray mode** lets you see through terrain to inspect underground track sections

## Terrain Editor

A dedicated terrain editor page provides tools for painting and sculpting the landscape.
