# Banana — Track System

The track system is the backbone of Banana. Every rail network starts with laying down tracks.

## Bezier Curve Editor

Tracks in Banana are drawn as interactive Bezier curves. The curve editor lets you place control points and adjust the shape of each segment with intuitive handles.

## Track Styles

Banana supports two visual track styles:

- **Ballasted** — traditional gravel bed tracks
- **Slab** — modern elevated or concrete slab tracks

## Electrification

Tracks can be electrified, rendering overhead catenary wires above the rails.

## Curve Tension

Each track segment has adjustable tension, controlling how smoothly the curve bends between control points.

## Track Operations

- **Splitting** — divide a track segment at any point
- **Deletion** — remove segments from the network
- **Junction management** — connect and disconnect track segments at junction points
